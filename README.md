# GQUPT
一款基于openai GPT api的windows问答软件，支持GPT3.5和GPT4， 无需翻墙真香<br>
使用方法：<br>
conda创建虚拟环境:  conda create -n GQUPT python=3.10  <br>
激活环境:           conda activate GQUPT <br>
下载安装包:         pip install requirements.txt  <br>
进入到GQUPT目录下运行 python main.py  <br>
